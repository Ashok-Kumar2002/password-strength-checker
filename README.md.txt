This is a simple Python-based Password Strength Checker designed to help users evaluate how secure their passwords are. It checks password strength based on multiple factors including length, character variety, and whether the password is commonly used.

# Features

- Checks password length (minimum 8 characters)
-  Validates presence of:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Detects commonly used (weak) passwords from a reference list
-  Provides a final rating: **Very Weak**, **Weak**, **Medium**, or **Strong**
